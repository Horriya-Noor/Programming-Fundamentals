## 🛠️ Core Topics Covered

Here is a breakdown of the fundamental concepts mastered throughout the semester:

### 1. Introduction to Programming & Logic
* Understanding Compiler, Interpreter, and Linker.
* Writing Pseudo-code and drawing Flowcharts.
* The anatomy of a basic program (Structure, Main function, Header files).

### 2. Variables, Data Types & Operators
* **Data Types:** `int`, `float`, `double`, `char`, `bool`.
* **Operators:** Arithmetic, Relational, Logical, and Assignment operators.
* Input/Output streams (`cin`/`cout` or `input()`/`print()`).

### 3. Conditional Statements (Decision Making)
* `if`, `else if`, and `nested if-else` structures.
* `switch-case` statements for multi-way branching.

### 4. Loops & Iterations
* `while` loops (Pre-test loops).
* `do-while` loops (Post-test loops).
* `for` loops (Counter-controlled loops).
* Handling nested loops and loop control statements (`break`, `continue`).

### 5. Functions & Modular Programming
* Function declaration, definition, and calling.
* Passing arguments: **Call by Value** vs. **Call by Reference**.
* Variable scope (Local vs. Global variables).

### 6. Arrays & Strings
* Single-Dimensional and Multi-Dimensional Arrays.
* Array manipulation (Searching and basic sorting).
* String handling and built-in string functions.

### 7. Pointers & Memory Management *(Introductory)*
* Understanding memory addresses (`&` operator).
* Pointer variables and dereferencing (`*` operator).
